# PoApper Homepage (Renewal)

PoApper club homepage

## Instructions

### Prerequisites
- Github Access token  
    - Example at `.env.example`  
- Node version >= 18.14.1  

### Development

```sh
$ npm install  
$ npm run dev  
```

### Building

```sh  
$ npm install  
$ npm run build  
```

## Technology Stack
### Front-end
- Astro
- TailwindCSS
